# 🚀 Smart Hackathon Management Platform

A scalable and intelligent hackathon platform inspired by Unstop, designed to streamline event management, participant collaboration, and project evaluation.

---

## 📌 Overview

The Smart Hackathon Management Platform enables:
- 🎯 Students to discover and participate in hackathons
- 🤝 Team formation based on skills
- 📤 Easy project submission
- 🧑‍💼 Organizers to manage events efficiently
- 📊 Real-time tracking and leaderboard updates

---

## ✨ Key Features

### 👤 User Features
- User Registration & Login (JWT/Firebase)
- Browse and search hackathons
- Join/Create teams
- Submit projects (GitHub/links/files)
- Track submission status

### 🧑‍💼 Organizer Features
- Create & manage hackathons
- Define rules, deadlines, and themes
- Evaluate submissions
- Announce results

### ⚡ Advanced Features
- 🔄 Real-time leaderboard (WebSockets)
- 🤖 AI-based project evaluation
- 📧 Automated email notifications
- 📄 Certificate generation (PDF)
- 💬 Team chat system (optional)
